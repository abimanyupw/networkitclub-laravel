# Network IT Club Laravel

Aplikasi manajemen atau portal web untuk Network IT Club yang dibangun menggunakan framework Laravel.

## Deskripsi Proyek

Proyek ini merupakan pengembangan ulang dari sistem Network IT Club menggunakan Laravel untuk memberikan performa yang lebih baik, kemudahan pemeliharaan, dan skalabilitas. Fokus utama proyek ini mencakup pengelolaan data anggota, konten klub, dan integrasi fitur-fitur pendukung lainnya.

## Teknologi yang Digunakan

- **Framework:** [Laravel 12](https://laravel.com/) (atau versi terbaru yang digunakan)
- **Frontend:** [Blade Templating Engine](https://laravel.com/docs/blade), [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Database:** MySQL / PostgreSQL (sesuaikan dengan yang Anda gunakan)

## Prasyarat

Pastikan perangkat Anda sudah terinstal:

- PHP >= 8.2
- Composer
- Node.js & NPM
- Database Server (MySQL/MariaDB)

## Instalasi

1. **Clone Repositori**

    ```bash
    git clone [https://github.com/abimanyupw/networkitclub-laravel.git](https://github.com/abimanyupw/networkitclub-laravel.git)
    cd networkitclub-laravel

    ```

2. **Instal library PHP**

    ```bash
    composer install
    ```

3. **Instal package JavaScript/CSS**

    ```bash
    npm install
    ```

4. **. Konfigurasi Lingkungan (.env)**

    ```bash
    cp .env.example .env
    ```

5. **Generate Key**
    ```bash
    php artisan key:generate
    ```
